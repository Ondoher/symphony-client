TempoUI Library

TempoUI is a library of web element styles based on Symphony's emerging Tempo Design Language. This library is independent, free from dependence on external libraries and frameworks. TempoUI aims to be truly framework agnostic, yet framework friendly.

Get it on GitHub

Run it: 

npm install
gulp serve

Use it:
include in your project any way you desire

Add to it: 

gulp watch
Change stuff -> Create PR -> We'll take care of the rest

Buttons

Button background and text color are based on sass variables.

Floating Action Button

<button class="tempo-btn tempo-btn--fab tempo-btn--primary">[button text]</button>

Flat Button

<button class="tempo-btn tempo-btn--flat">[button text]</button>

Raised Button

<button class="tempo-btn tempo-btn--raised tempo-btn--primary">[button text]</button>

Button Link

<button class="tempo-btn tempo-btn--link">[button text]</button>

Cancel/Dismiss Button Link

<button class="tempo-btn tempo-btn--dismiss">[button text]</button>

Other Variations

for compact mode 
<button class="tempo-btn tempo-btn--raised tempo-btn--mini tempo-btn--secondary">[button text]</button>
warning variation 
<button class="tempo-btn tempo-btn--raised tempo-btn--warning">[button text]</button>
this is diabled, see 
<button class="tempo-btn tempo-btn--raised tempo-btn--disabled" disabled>[button text]</button>
